# Study Planner Web App

A professional multi-page Study Planner Web Application built using HTML, CSS, and JavaScript.

## Features

- Multi-page dashboard architecture
- Task management (Add, Delete, Complete)
- Task filtering (All / Completed / Pending)
- Dashboard analytics with progress tracking
- Pomodoro Timer (Study / Break mode)
- Dark Mode toggle with persistence
- LocalStorage data persistence

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Transitions)
- JavaScript (DOM Manipulation, LocalStorage)
